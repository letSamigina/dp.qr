Overview
  - Minecraft: Java Edition datapack, pack_format 101.1
  - Version 3 QR code, error-correction level M
  - Draws by placing blocks on a 128×128 grid at your run position—the area the smallest map can depict

How to use
  1. Add this pack to the world.
  2. Open Quick Actions (default: G). Open the [.drc.qr] menu.
  3. Enter a URL. Optional: reserve space for player-head icons (you place heads yourself).
  4. Run. If the game warns before running the command, that is normal; continue.
  5. You may get a map if you have none.

Limits
  - URL max length: 42 characters (fixed). Longer input shows an error and will not render.
